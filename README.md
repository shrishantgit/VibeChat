# VibeChat

VibeChat is a full-stack, real-time messaging application. It features secure authentication, real-time communication, media sharing, and a fully customizable UI with multiple themes and wallpapers.

## 🚀 Features
* **Real-time Messaging:** Instant message delivery using Socket.io.
* **Authentication:** Secure user login and registration powered by Clerk.
* **Media Uploads:** Send images and videos optimized by ImageKit.
* **Customizable UI:** Toggle between light/dark modes and apply 11 different themes and 13 wallpapers.
* **Interactive Experience:** Real-time online/offline indicators and satisfying keyboard typing sounds.

## 💻 Tech Stack
* **Frontend:** React 19, Tailwind CSS 4, Hero UI, Zustand (State Management)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB, Mongoose
* **Real-time:** Socket.io
* **Media & Auth:** ImageKit, Clerk

## 🛠️ Local Development Setup

### Prerequisites
Make sure you have Node.js and Docker installed on your machine.
